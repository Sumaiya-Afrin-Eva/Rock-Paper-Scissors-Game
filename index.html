Select One!

🎮 How The Game Works:

1. 👥 Two players join the same room by entering a shared Room Code and selecting their roles (Player 1 or Player 2).

2. Player 1 always starts with “Your Turn”☺️, Player 2 sees “Opponent’s Turn😒.”

3. 📨 Each player selects a move: Rock, Paper, or Scissors.

4. After both players have submitted, they can see their own result.🏆

📄 Rules:

✊Rock beats Scissors✌️

✌️Scissors beats Paper✋

✋Paper beats Rock✊

🗣️: Enter a random room code and select your role.
